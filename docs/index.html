About

safe-clean is a powerful yet safe disk cleanup tool that helps you identify and remove unnecessary files from your system. Built with Rust for performance and safety, it provides both an interactive TUI (Terminal User Interface) and a comprehensive CLI for all your cleanup needs.

Features

üé® Interactive TUI Mode

Browse and explore your disk usage with an intuitive terminal interface that makes cleanup easy and visual.

üê≥ Docker Cleanup

Safely remove unused Docker containers, images, volumes, and networks to reclaim disk space.

üóëÔ∏è Temporary Files

Clean system temporary directories and files that accumulate over time.

üìä Directory Analysis

List directories by size to quickly identify where your disk space is being used.

üîç Large File Discovery

Find files larger than a specified threshold to identify space hogs.

üíª Dev Artifacts Cleanup

Discover and remove node_modules, .venv, target, and other development artifacts.

üîí Dry Run Mode

Preview what would be cleaned without actually removing anything - safety first!

‚úÖ Safety Checks

All operations include confirmation prompts and safety checks to prevent accidents.

Installation

Build from Source

# Clone the repository
git clone https://github.com/npsg02/safe-clean.git
cd safe-clean

# Build the release version
cargo build --release

# The binary will be available at:
# ./target/release/safe-clean

Usage

Interactive TUI Mode (Default)

Launch the interactive terminal interface for the best user experience:

safe-clean
# or explicitly
safe-clean tui

Command Line Interface

Use CLI commands for scripting and automation:

CLI Commands

üìã List Directories by Size

# List top 20 largest directories in current directory
safe-clean list

# List top 10 largest directories
safe-clean list --top 10

# Analyze specific path
safe-clean list /path/to/analyze --top 15

üîç Find Large Files

# Find files larger than 100MB (default)
safe-clean large

# Find files larger than 1GB
safe-clean large --size 1GB

# Search in specific directory
safe-clean large /path/to/search --size 500MB

üíª Development Artifacts Cleanup

# Find development artifacts (dry run)
safe-clean dev-clean --dry-run

# Actually remove development artifacts (with confirmation)
safe-clean dev-clean

# Search in specific directory
safe-clean dev-clean /path/to/projects --dry-run

üê≥ Docker Cleanup

# Preview Docker cleanup
safe-clean docker --dry-run

# Clean up Docker resources (with confirmation)
safe-clean docker

üóëÔ∏è Temporary Files Cleanup

# Preview temp files cleanup
safe-clean temp --dry-run

# Clean up temporary files (with confirmation)
safe-clean temp

‚öôÔ∏è Command Options

Option Description
--dry-run Preview what would be cleaned without actually removing anything
--top N Limit results to top N items (for list command)
--size SIZE Specify size threshold (e.g., "100MB", "1GB", "500KB")

Safety Features

üõ°Ô∏è Built-in Safety Mechanisms

  • Confirmation Prompts: All destructive operations require explicit user confirmation
  • Dry Run Mode: Preview operations before executing them to see exactly what will be affected
  • Safe File Detection: Only removes files that match known safe patterns
  • Path Validation: Prevents removal of system-critical directories
  • Detailed Reporting: Shows exactly what will be or was cleaned with size information

Development Artifacts Detected

  • node_modules - Node.js dependencies
  • .venv, venv - Python virtual environments
  • __pycache__ - Python cache files
  • .tox - Python testing artifacts
  • target - Rust build artifacts
  • build, dist - General build artifacts

Size Format

Size values can be specified in various formats:

  • 1024 - bytes
  • 1KB, 1MB, 1GB, 1TB
  • 1.5GB, 500MB, etc.
